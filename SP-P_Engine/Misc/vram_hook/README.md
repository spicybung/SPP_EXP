# VRAM Extension Hook

Этот модуль ядра (PRX) предназначен для разблокировки всех 4MB VRAM на PSP Slim и выше.

## Особенности сборки

1. **export.c** Удалить export.c в папке с хуком, иначе будут ошибки линковки, о которых компилятор промолчит. Возможно связано с нерегулируемой версионностью инструмента.

2. **Прототипы (C++)** Если исходники C++, то прототипировать функцию там, где она будет использоваться (хук написан на си):
```cpp
   extern "C"
{
    void VramSetSize(int kb);
}
//Иначе (если исходники на си): прописываем хедер vram.h:
#include "путь\vram.h"

3. 4 библиотеки прекомпилировны в libs/precompiled. Подходят, оставить там.

4. Папка с инклюдами и доп. либами есть, но они не используются. Инклайдить kubridge.h в исходник не обязательно, у меня работает и без него.

5. В мейкфайл ВАШЕГО исходника (а не хука), нужно добавить VramExt.o в OBJS секцию. Путь там, где лежит VramExt.S

Порядок сборки хука:
1. Запускаем BUILDSTUB.bat
2. Запускаем BUILD.bat
3. Прописываем прототип VramSetSize, прописываем сборку объектника в мейкфайл проекта, переносим vramext.prx в нужную папку(п.4).
4. Прописываем в исходник проекта, старт модуля:

```cpp
//тут стартуем внешний кернел модуль
VRAM_hook = pspSdkLoadStartModule(VRAM_module, PSP_MEMORY_PARTITION_KERNEL);
//Вызываем расширение до 4096kb (4mb) 
VramSetSize(4096); 

ГДЕ VRAM_module - путь до модуля. Если он в корне с EBOOT.PBP, то просто vramext.prx
